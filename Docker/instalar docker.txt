Paso a paso para instalar wsl

1. Abrir PowerShell como administrador: Win+X y elegir PowerShell como admin.

2. Pegar estos comandos uno por uno:

	1. dism.exe /online /enable-feature /featurename:Microsoft-Windows-Subsystem-Linux /all /norestart

	2. dism.exe /online /enable-feature /featurename:VirtualMachinePlatform /all /norestart

	3. Reiniciar el PC

3. Abrir PowerShell como administrador: Win+X y elegir PowerShell como admin.

4. Instalar WSL con este comando:
wsl --install -d Debian

5. Asegurar la ultima version y la arquitectura:
wsl --set-version Debian 2

6. Abrir y configurar Debian

7. Verificar las versiones:
sudo apt update && sudo apt upgrade -y

8. Instalar dependencias:
sudo apt install -y ca-certificates curl gnupg

9. Añadir llave GPG de Docker:
sudo install -m 0755 -d /etc/apt/keyrings

curl -fsSL https://download.docker.com/linux/debian/gpg | sudo gpg --dearmor -o /etc/apt/keyrings/docker.gpg

sudo chmod a+r /etc/apt/keyrings/docker.gpg

10. Configurar el repositorio de Docker:
echo "deb [arch=$(dpkg --print-architecture) signed-by=/etc/apt/keyrings/docker.gpg] https://download.docker.com/linux/debian $(. /etc/os-release && echo "$VERSION_CODENAME") stable" | sudo tee /etc/apt/sources.list.d/docker.list > /dev/null

11. Instalar paquete de Docker:
sudo apt update && sudo apt install -y docker-ce docker-ce-cli containerd.io docker-buildx-plugin docker-compose-plugin

12. Configuracion de usuario:
sudo usermod -aG docker $USER

13. Cerrar y abrir Debian

14. Iniciar Docker (se debe hacer cada que quiera usar docker):
sudo service docker start
